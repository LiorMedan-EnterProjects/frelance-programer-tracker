# Agent Rules & Preferences

## Package Manager
- **ALWAYS use `pnpm`** for all package management and script execution tasks.
- Do NOT use `npm` or `yarn`.

## Language & Locale
- The main language of the application is **Hebrew** (he).
- Layout direction is **RTL** (Right-to-Left).

## Design System
- **Theme**: Premium Dark/Light mode (Default: Dark).
- **Styling**: Material UI with Emotion (`stylis-plugin-rtl`).
- **Icons**: Material UI Icons.
